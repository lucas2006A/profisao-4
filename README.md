# 🌾 MT vs PR Agro - Comparação Agronegócio

Um site elegante e interativo que compara o agronegócio entre **Mato Grosso** e **Paraná**, com dados de produção, preços de commodities, salários e um assistente de IA integrado.

## 🎯 Funcionalidades

- **Comparação Visual** - Dados lado a lado de MT vs PR
- **Preços de Commodities** - Soja, Milho, Trigo e Algodão com gráficos de evolução
- **Análise de Salários** - Comparação de remuneração por categoria profissional
- **Dados de Produção** - Gráficos de produção e composição por commodity
- **Sistema de Comentários** - Comunidade comentando sobre os dados
- **Assistente IA** - Chat inteligente para responder dúvidas sobre dados agrícolas

## 📊 Dados Principais

### Mato Grosso
- **Produção Total (2025/26)**: 48,5 milhões de toneladas
- **Preço Soja**: R$ 107,19 por saca
- **Salário Trabalhador Rural**: R$ 2.473/mês
- **Participação Nacional**: ~55% da produção de grãos

### Paraná
- **Produção Total (2025/26)**: 22,3 milhões de toneladas
- **Preço Soja**: R$ 117,00 por saca
- **Salário Agropecuária**: R$ 3.428/mês
- **Participação Nacional**: ~13,9% da produção de grãos
